Build and run with using Ant

Change the following "property name="client_file_codebase" location="/cs/student/pazspm/public_html/cs290b/hw1/client/" " inside build.xml to your public_html folder

//Compile classes

ant compile

//Create client and server jar files

ant dist

//Create javadocs

ant javadoc

ant runComputer

ant runClientTsp

ant runClientMandelbrot

Compiling and running without Ant

// Compiling api and creating compute.jar

javac api/*.java

jar cvf compute.jar compute\*.class


// Compiling and running server side (system)

javac system/Main.java

java -Djava.security.policy=Security.policy system/Main


// Compiling tasks

javac tasks/*.java


// Compiling and running clients

javac tasks/*.java

java -Djava.security.policy=Security.policy client/ClientMandelbrotSet

java -Djava.security.policy=Security.policy client/ClientEuclideanTsp
